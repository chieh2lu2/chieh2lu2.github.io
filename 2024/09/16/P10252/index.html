P10252 题解

一直没有想到 aaa 可能等于 111，白交了好多发，好蠢……

题意

题面很清楚了。

解析

有一个很且显然的想法是：每次 xxx 变换后，如果变换后的 x′>xx ^ \prime > xx′>x，那么 xxx 在接下来的变换里只会越来越大；反之亦然。具体的下面会提及。

找找规律：

变换一次：x1=ax−bx_1 = ax - bx1​=ax−b。
变换两次：x2=ax1−b=ax2−ab−bx_2 = ax_1 - b = ax^2-ab-bx2​=ax1​−b=ax2−ab−b。
变换三次：x3=ax2−b=ax3−a2b−ab−bx_3 = ax_2 - b = ax^3-a^2b-ab-bx3​=ax2​−b=ax3−a2b−ab−b。
变换 iii 次：xi=aix−∑j=0i−1ajbx_i = a^ix - \sum_{j=0}^{i-1}a^jbxi​=aix−∑j=0i−1​ajb。

嘶……没头绪。那看看每次 xxx 增加的量是多少：

\begin{aligned}

\Delta x &= x_i-x_{i-1} \
&= a^ix - \sum_{j=0}{i-1}ajb - \Bigg( a^{i-1}x - \sum_{j=0}{i-2}ajb \Bigg) \

&= aix-a{i-1}x-a^{i-1}b \
&= a^{i-1}(ax - x + b)

\end{aligned}

可以看见 ax−x+bax - x + bax−x+b 这个数是第一次变换后，x1x_1x1​ 与 xxx 的差值。xxx 的增加与否与前面的 ai−1a^{i-1}ai−1 无关（a≥0a \ge 0a≥0），只跟 ax−x+bax - x + bax−x+b 有关，而 ax−x+bax - x + bax−x+b 是一个定值。这也说明了开头猜想的正确性。

从这里我们可以发现一种特殊情况：如果 ax−x+b≥0ax - x + b \ge 0ax−x+b≥0，那么 xxx 变换后不可能变小，一开始的 xxx 就是最小值。这种情况直接输出 xxx 即可。

现在来考虑 的一般情况。我们要求记的毕竟不是增加量，而是想求出它每一次变换后 xix_ixi​ 的大小。使用上面的差值求和试试看？记 ax−x+bax - x + bax−x+b 为 ddd，设 iii 为变换次数，那么变换 iii 次后：

xi=x+∑Δx=x+∑i=1ai−1d=x+d×ai−1a−1x_i =x+\sum \Delta x = x + \sum_{i=1} a^{i-1} d = x + d \times \frac{a^i-1}{a-1} xi​=x+∑Δx=x+i=1∑​ai−1d=x+d×a−1ai−1​

那么只要枚举 iii，判断这个能在右边的式子是否小于 000 即可。Δx\Delta xΔx 中的 ai−1a^{i-1}ai−1 增长速度很快（a≤109a \le 10^9a≤109），不用二分就能过（或许可能数据太水了 = =）。

注意，这里的 a−1a - 1a−1 为分母，需要特判 a=1\pmb{a = 1}a=1a=1 的情况。答案易知是 x−⌈xb⌉×bx - \lceil \frac{x}{b} \rceil \times bx−⌈bx​⌉×b。还有一种特殊情况：当 a=1a = 1a=1 或 x=1x = 1x=1 时，直接输出 −b-b−b 即可。
